## 1. 数据库说明

- 数据库类型：MySQL 8.0
- 字符集：utf8mb4

## 2. 数据表设计

### 2.1 司机表（sj_siji）

| 字段名 | 类型 | 长度 | 允许NULL | 默认值 | 说明 |
|--------|------|------|----------|--------|------|
| sj_id | INT | - | NO | AUTO_INCREMENT | 主键ID |
| sj_bianhao | VARCHAR | 20 | NO | - | 司机编号（唯一） |
| sj_xingming | VARCHAR | 50 | NO | - | 姓名 |
| sj_shouji | VARCHAR | 20 | NO | - | 手机号（唯一） |
| sj_mima | VARCHAR | 100 | NO | - | 密码（加密） |
| sj_shenfenzheng | VARCHAR | 20 | YES | NULL | 身份证号 |
| sj_chepaihao | VARCHAR | 20 | YES | NULL | 车牌号 |
| sj_zhuangtai | INT | - | NO | 1 | 状态（1正常 0禁用） |
| sj_ruzhi_shijian | DATETIME | - | YES | NULL | 入职时间 |
| sj_beizhu | TEXT | - | YES | NULL | 备注 |
| sj_chuangjian_shijian | DATETIME | - | NO | CURRENT_TIMESTAMP | 创建时间 |
| sj_gengxin_shijian | DATETIME | - | NO | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

**索引：**
- PRIMARY KEY: sj_id
- UNIQUE KEY: sj_bianhao
- UNIQUE KEY: sj_shouji
- INDEX: sj_zhuangtai

```sql
CREATE TABLE sj_siji (
  sj_id INT PRIMARY KEY AUTO_INCREMENT,
  sj_bianhao VARCHAR(20) UNIQUE NOT NULL COMMENT '司机编号',
  sj_xingming VARCHAR(50) NOT NULL COMMENT '姓名',
  sj_shouji VARCHAR(20) UNIQUE NOT NULL COMMENT '手机号',
  sj_mima VARCHAR(100) NOT NULL COMMENT '密码',
  sj_shenfenzheng VARCHAR(20) COMMENT '身份证号',
  sj_chepaihao VARCHAR(20) COMMENT '车牌号',
  sj_zhuangtai INT DEFAULT 1 COMMENT '状态1正常0禁用',
  sj_ruzhi_shijian DATETIME COMMENT '入职时间',
  sj_beizhu TEXT COMMENT '备注',
  sj_chuangjian_shijian DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  sj_gengxin_shijian DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  INDEX idx_sj_zhuangtai (sj_zhuangtai)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='司机表';
```

| sj_caozuo_shijian | DATETIME | - | NO | CURRENT_TIMESTAMP | 操作时间 |

**索引：**
- PRIMARY KEY: sj_id
- INDEX: sj_caozuo_ren
- INDEX: sj_caozuo_shijian
