## 已实现功能

✅ 订单创建（含订单号唯一性验证）
✅ 订单列表查询（分页、状态筛选、司机/乘客筛选）
✅ 订单详情查询（含关联的司机和乘客信息）
✅ 订单信息更新
✅ 订单状态管理（待接单、进行中、已完成、已取消）
✅ 司机指派
✅ 所有接口需要身份验证（Token）
✅ 统一的响应格式和错误码
