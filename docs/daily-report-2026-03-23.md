## 💬 需要用户配合的事项

### 1. 安装 MySQL 数据库

**方案 A: Docker（推荐）**
```bash
docker run --name changanxing-mysql \
  -e MYSQL_ROOT_PASSWORD=your_password \
  -e MYSQL_DATABASE=changanxing \
  -p 3306:3306 \
  -d mysql:8.0
```

**方案 B: 直接安装**
- 下载地址：https://dev.mysql.com/downloads/mysql/
- 执行初始化脚本：`mysql -u root -p < docs/database-admin-init.sql`

### 2. 配置环境变量

```bash
# 复制 .env.example 为 .env.local
cp .env.example .env.local

# 修改数据库配置
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=你的密码
DB_NAME=changanxing

# JWT 密钥（随机字符串）
JWT_SECRET=your-super-secret-jwt-key
```
